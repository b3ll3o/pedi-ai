Stack Tecnológico

Next.js 16 + TypeScript React 19 NestJS + Fastify Prisma ORM + PostgreSQL Socket.io Service Worker + IndexedDB Zustand + React Query MercadoPago PIX Stripe

Entry Points

Rotas Públicas

  • / Landing page
  • /restaurantes Lista restaurantes
  • /restaurantes/[id]/cardapio Cardápio digital

Rotas Cliente

  • /menu Cardápio legado
  • /cart Carrinho
  • /checkout Checkout

Rotas Admin

  • /admin/dashboard Painel admin
  • /admin/orders Pedidos
  • /kitchen KDS Display

API Routes

  • /api/auth/* Autenticação
  • /api/admin/* Admin APIs
  • /api/payments/* Pagamentos
  • /api/webhooks/* Webhooks

Socket.io Events (Realtime)

Client → Server

  • joinRestaurant Entrar room
  • leaveRestaurant Sair room

Server → Client

  • newOrder Novo pedido
  • orderUpdate Status atualizado

Rooms

  • restaurant:{id} Sala do restaurante

Hooks de Realtime

useSocketIO

  • Abstração de conexão Socket.io
  • Auto-reconexão com backoff
  • Gestão de rooms

useRealtimeOrders

  • Admin dashboard
  • Escuta orderUpdate + newOrder
  • Polling fallback (10s)

usePedidosKDS

  • Kitchen Display System
  • Som ao receber pedido
  • Polling fallback (5s)

useCustomerOrderNotifications

  • Cliente monitora pedido
  • Escuta orderUpdate
  • Polling fallback (5s)
