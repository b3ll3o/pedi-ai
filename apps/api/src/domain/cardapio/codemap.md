# Domain Layer — Cardápio

> Bounded Context de cardápio (categorias, produtos, modificadores, combos).

## Entidades

- `Categoria` — Categorias do cardápio
- `Produto` — Produtos individuais
- `GrupoModificador` — Grupos de modificadores (ex: "Tamanho")
- `ValorModificador` — Valores dentro de um grupo (ex: "Pequeno", "Médio", "Grande")
- `Combo` — Combos com preço fixo
- `ItemCombo` — Itens de um combo
