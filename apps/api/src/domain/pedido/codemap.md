# Domain Layer — Pedido

> Bounded Context de pedidos e itens.

## Entidades

- `Pedido` — Pedido com status e totais
- `ItemPedido` — Item individual do pedido
- `HistoricoStatusPedido` — Histórico de mudanças de status

## Value Objects

- `StatusPedido` — Status do pedido (pending_payment, paid, preparing, ready, delivered, cancelled)
